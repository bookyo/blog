## 4. PDF Table Layout Engine

**Use it when:** You need to share structured data — inventory lists, financial reports, audit logs — in a format that actually looks presentable.

**Key features:**
- Auto pagination with row keep-together logic
- Repeating headers across pages
- Zebra row styling for readability
- JSON or CSV input with optional column order override

**Try it:** [PDF Table Layout Engine →](https://elysiatools.com/en/tools/pdf-table-layout-engine)

## 6. HTML to PDF Renderer

**The problem:** Sometimes you need full HTML/CSS control. Sometimes you just need a reliable, deterministic conversion without a full browser.

This tool has two modes. The subset renderer uses PDFKit — it's fast and deterministic, perfect for server-side generation. The browser renderer fires up Puppeteer for full CSS support, including things like `@page` rules and `print` media queries.

**Use it when:** You're building automated report generation, converting HTML emails to PDF, or need pixel-perfect output for legal documents.

**Key features:**
- Two rendering modes: fast PDFKit subset or full Puppeteer browser
- Configurable page size (A4, Letter, Legal), margins, and orientation
- Background color/image printing option
- CSS media type support via browser mode

**Try it:** [HTML to PDF Renderer →](https://elysiatools.com/en/tools/html-to-pdf-renderer)
