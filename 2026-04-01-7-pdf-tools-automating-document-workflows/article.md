## 1. PDF Invoice Generator

The tool that replaces "download template → fill in fields → export PDF."

You feed it JSON line items, company info, a logo, and a tax rate. It outputs a clean, print-ready A4 or Letter invoice with a subtotal, tax line, and total — automatically formatted.

```
[{"description":"Design Sprint","quantity":3,"unitPrice":1200}]
```

Input your company name, customer name, invoice number, and due date. The tool handles the rest, including currency formatting for USD, EUR, GBP, and more.

This is the kind of tool a freelancer or small agency can bookmark and use every week.

**Use it:** [PDF Invoice Generator](https://elysiatools.com/en/tools/pdf-invoice-generator)

## 2. PDF Receipt / Ticket Generator

Thermal receipt printers need narrow-format PDFs. This tool generates exactly that — 58mm or 80mm width, with optional QR codes for digital receipt links.

You input items as JSON, set the store name and tax rate, and it outputs a compact, monospace receipt with a dashed separator line and a totals block.

The QR code field accepts any URL or text — perfect for linking to an order confirmation page or a digital version of the receipt.

**Use it:** [PDF Receipt / Ticket Generator](https://elysiatools.com/en/tools/pdf-receipt-ticket)

## 3. PDF Certificates Batch

One certificate is easy. 80 certificates for a training cohort is a different problem.

This tool takes a CSV — name, date, course, score per row — and generates one PDF with one certificate page per row. You control the theme color, the organization name, the award text, and the signature line.

Paste in:

```
name,date,course,score
Alice Zhang,2026-02-01,TypeScript Fundamentals,98
Bob Chen,2026-02-01,TypeScript Fundamentals,95
```

You get a multi-page PDF, each page a proper certificate. HR and training teams, start bookmarking.

**Use it:** [PDF Certificates Batch](https://elysiatools.com/en/tools/pdf-certificates-batch)

## 4. PDF Grade Report Card

Schools and tutoring services face the same document problem at scale: one report card per student, per term, with subjects, scores, teacher comments, and class rankings.

This tool takes structured JSON — student ID, class, attendance, subjects with scores, and comments — and generates a multi-page PDF with one report card per student. Each card includes a subject table, a totals row, and an overall comment section.

The output is formatted to look like something you'd receive from a school, not something you'd print from a spreadsheet.

**Use it:** [PDF Grade Report Card](https://elysiatools.com/en/tools/pdf-grade-report-card)

## 5. PDF Sales Dashboard

Meeting-ready reports with charts, KPI cards, and data tables in a single PDF — that was a full design sprint last year.

Now it's a form. Input your report title, period, currency, and metric cards (Revenue, Orders, Conversion, AOV — with delta percentages). Feed it a JSON sales table and optional chart images. The tool assembles a branded, landscape PDF with a dark header, metric tiles, chart blocks, a data table with totals, and a conclusions section.

This means data teams can generate a fresh PDF before every review meeting in under two minutes.

**Use it:** [PDF Sales Dashboard](https://elysiatools.com/en/tools/pdf-sales-dashboard-pdf)

## 6. PDF Payroll Slip Batch

Payroll software exists. For one-off situations — contractor payments, small team bonuses, final settlements — opening payroll software feels like overkill.

This tool takes JSON payroll data per employee and outputs either a merged multi-page PDF or a ZIP of individual slips. Each slip shows earnings, deductions, total deductions, and net pay in a clean card layout.

Output as one merged PDF for record-keeping, or as separate files to send individually to employees.

**Use it:** [PDF Payroll Slip Batch](https://elysiatools.com/en/tools/pdf-payroll-slip-batch)

## 7. PDF Watermark Tool

Once you've generated and sent a document, the last step is protecting it. This tool adds text or shape watermarks to any existing PDF — document-level (every page) or page-level (specific pages you choose).

Options include rotation angle, opacity, font size, color, and a tiling mode that scatters the watermark across the page diagonally. The most common use is a CONFIDENTIAL text watermark on the first page of a financial report before distribution.

**Use it:** [PDF Watermark Tool](https://elysiatools.com/en/tools/pdf-watermark-tool)
