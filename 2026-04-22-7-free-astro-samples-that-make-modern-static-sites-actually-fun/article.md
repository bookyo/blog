## Why This Matters

The static site landscape has fundamentally shifted. It's no longer about "static HTML pages." Today's static sites need:

- **Partial hydration** — Only load JavaScript when and where it's needed
- **Multi-framework support** — Use React for interactive widgets, Vue for forms, and vanilla JS for the rest
- **Content collections** — Type-safe content that scales without CMS overhead
- **Edge deployment** — Ship to Cloudflare Workers, Vercel Edge, or Netlify in minutes

Astro delivers all four. These samples give you the patterns to actually use them.
